オレキシン作動薬を DSL中間表現型 で説明

「自然文 → DSL → レンダリング」の流れで、 オレキシン作動薬（OX2R活性化による覚醒維持）の仕組みを、 nodes/edges JSON標準とMermaid系記法思想で可視化します。

1) 自然文 → DSL → レンダリング

自然文 → DSL（中間表現） → レンダリング

2) 自然文（入力例）

短文の知識記述を、機械処理しやすい構造へ落とし込みます。


      


      

        
3) DSL（中間表現）

        
因果関係を「主語 / 関係 / 目的語 / 効果極性」で表現します。

        

      


      

        
4) nodes / edges JSON標準

        
UI描画や他システム連携向けの標準的グラフJSONです。

        

      


      

        
5) Mermaid系記法思想（テキストで図を記述）

        
読みやすい一行関係を重ねて、構造を明示します。

        

      


      

        
6) レンダリング結果（簡易テキストビュー）

        
Mermaid実行環境がなくても、関連を順に追える表示です。
