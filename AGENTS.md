# AGENTS.md

このリポジトリは、ユーザーが AI に依頼して作成したさまざまな Web サイトを配置し、
一時的にホストするための場所です。

## 目的と構成
- ルートの `index.html` は**全体案内ページ**として使います。
- `index.html` には複雑な実装を置かず、各サブディレクトリへのリンク一覧を掲載してください。
- 個別のテーマごとのサイトは、サブディレクトリ配下に作成してください。
- `index.html` から各サブディレクトリのページへ遷移できる構成にしてください。

## 実装ルール
- **TypeScript の使用は禁止**です。
- フロントエンド実装は、基本的に **HTML と JavaScript** で行ってください。

## セキュリティ・プライバシー
- ユーザーの個人情報（PII）は、**絶対に出力・表示・記録しない**でください。
- ログ、サンプルデータ、コメント、デバッグ出力にも個人情報を含めないでください。

## 補足
- 追加するページは、後から一覧しやすいようにディレクトリ名・ページ名を分かりやすくしてください。
- 必要最小限で保守しやすい構成を優先してください。
